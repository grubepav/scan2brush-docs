# Scan2brusH™ – Dental Workflow Module

Welcome to the online documentation for the **Scan2brusH™ Dental Workflow Module**.

Scan2brusH is a Blender add-on that provides a streamlined dental workflow:
importing dental scans, preparing geometry and exporting data for lab workflows.

## What you need

- **Blender 4.0.1** (desktop application)
- **Scan2brusH add-on** – installed as a standard Blender add-on (`.zip` file)

If you only want to get started quickly, follow the **[Quick Installation](install-quick.md)**.
For a step‑by‑step guide with screenshots, see **[Detailed Installation](install-detailed.md)**.
